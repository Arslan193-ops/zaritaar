# Zaritaar

This repository contains the complete codebase for the Zaritaar store, including:

- **frontend/**: A Next.js application for the storefront.
- **mystore/**: A Sanity project for content management.

## Deployment

The frontend is designed to be deployed on **Vercel**.

### Prerequisites

- Node.js installed locally.
- A Vercel account linked to your GitHub.

### Getting Started

1. Clone the repository.
2. Install dependencies in each subdirectory or at the root if using workspaces.
3. Set up environment variables in `.env` files (see `.env.example` if available).
